# Features

## Authentication

Authentication is handled by [Auth.js](https://authjs.dev/reference/libraries/nextjs-auth-js).

## Database

The database uses [Drizzle](https://orm.drizzle.team/docs/introduction) ORM.

## UI

[Shadcn](https://ui.shadcn.com/docs/components) built-in.

The Bones CLI can be used to generate components, you can find it at the `/cli` route.
